Principio de Bernoulli, caída libre de cuerpos a diferentes temperaturas

J F Juez, J. L. Navarro, A. E. Jara

Resumen


<!-- @page { margin: 2cm } P { margin-bottom: 0.21cm } -->

De acuerdo con la teoría clásica de Bernoulli, cuerpos iguales a temperatura distinta caen a velocidad diferente, debido al cambio de presión que se produce en el aire que rodea al cuerpo esférico que cae. Con el ánimo de comprobar este postulado, en el presente artículo se muestra un análisis experimental por medio del cual se da explicación a la veracidad del principio de Bernoulli, según el cual, el tiempo de caída libre de un cuerpo esférico con una temperatura inicial (T1 >> 0) cae más rápido que otro cuerpo de similares características pero con una temperatura inicial (T2 > 0) donde T1 >> T2. Para dar explicación a este fenómeno se utilizó la interface COBRA 3 (PHYWE), con un sensor de movimiento adecuado para realizar mediciones de objetos en caída libre, con un contador de seis décadas se midió el tiempo de caída para cada uno de los cuerpos, a partir del cual se realizó la contrastación correspondiente entre los resultados, para luego compararlos con la teoría mencionada y comprobar su veracidad.
